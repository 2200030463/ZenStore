# ZenStore 🛒

🔗 Live Demo: https://your-vercel-link.vercel.app
## 🚀 About Project
ZenStore is a full-stack E-Commerce web application built using React.js, Node.js, Express.js, and MongoDB. It allows users to browse products, add items to cart, and make secure payments using Razorpay.
## ✨ Features
- Product browsing
- Add to cart
- Wishlist functionality
- Secure payment (Razorpay)
- Order management
- Responsive design
- ## 🛠 Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Payment: Razorpay
- Deployment: Vercel
